# MineGraph Experiment Log

Each experiment gets a numbered entry in `experiments/`. Format:

```
experiments/
├── README.md          ← this file
├── E001.md            ← first experiment
├── E002.md            ← second experiment
└── ...
```

Each entry records: date, config, results, analysis, and decisions made.

Use `./scripts/analyze_experiment.sh logs/fleet-*` for raw data, then
summarize findings here.
